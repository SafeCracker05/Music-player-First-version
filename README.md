# Spotify Music Player

Welcome to the Spotify-inspired Music Player implemented in Python using Tkinter and Pygame! This simple music player allows you to play, pause, stop, and navigate through your music collection.

## Instructions

1. **Clone the Repository:**
   - Clone this repository to your local machine using the following command:
     ```bash
     git clone https://github.com/your-username/spotify-music-player.git
     ```
   - Replace `your-username` with your GitHub username.

2. **Navigate to the Project:**
   - Change your working directory to the cloned repository:
     ```bash
     cd spotify-music-player
     ```

3. **Install Dependencies:**
   - Make sure you have Python and Pygame installed on your machine.
     ```bash
     pip install pygame
     ```

4. **Run the Music Player:**
   - Execute the following command to start the Spotify Music Player:
     ```bash
     python music_player.py
     ```

5. **Customize Your Playlist:**
   - Update the `rootpath` variable in the code to the path containing your music files.
   - Ensure that your music files have the `.mp3` extension.

6. **Controls:**
   - **Play:** Click the "Play" button to start playing the selected song.
   - **Pause/Play:** Click the "Pause" button to pause or resume the current song.
   - **Stop:** Click the "Stop" button to stop the playback.
   - **Previous/Next:** Use the "Prev" and "Next" buttons to navigate through your playlist.

7. **Enjoy Your Music!**

## Supported File Format
The music player supports files with the `.mp3` extension.
